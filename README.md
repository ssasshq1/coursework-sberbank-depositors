# Система учета вкладчиков банка "Сбербанк"

## Описание
Консольное приложение для учета сведений о вкладчиках банка. Система предназначена для сотрудников банка, для управления данными о клиентах и их вкладах.

## Технологии
*   Язык: C++
*   Компилятор: g++ (MinGW)
*   Среда разработки: Visual Studio Code

## Запуск проекта
1. Клонируйте репозиторий.
2. Перейдите в папку `src`.
3. Выполните команду компиляции: `g++ *.cpp -o ../app`
4. Вернитесь в корневую директорию и запустите `./app` (или `.\app.exe` на Windows).
